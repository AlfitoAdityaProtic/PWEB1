Praktikum Pemrograman WEB

Praktikum Pemrograman Web membreikan Ketrampilan kepada mahasiswa dalam pembuatan situss web yang dinamis dan dapat berinteraksi dengan user. Materi meliputi: Teknologi Web, Pembuatan situs, HTML, CSS, PHP, MySQL dan Visual Studio Code. "E-learning" link.

Keterangan:

  1. Warna border adalah yellowgreen
  2. Gunakan transformasi teksuppercase pada heading
  3. Paragraf utama menggunakan indentasi dan rata kanan kiri
  4. Gunakan letter-spacing pada paragraf utama
  5. Warna teks hyperlink adalah #008CBA
  6. Pada saat link di klik maka akan membuka halaman https://elearning.pnc.ac.id/ pada Tab Baru
